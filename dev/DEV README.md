## Document Generation

### Resume Agent

The `/agents` folder contains AI agent configurations for generating targeted professional documents. The resume agent can produce audience-specific resumes incorporating:

- Professional credentials and experience
- Publication abstracts with relevance statements
- Alignment with SER Best Practices and PIABA standards

### Target Audiences

- **PIABA/SER** — Investment fraud attorneys and expert witness colleagues
- **Retaining Counsel** — Attorneys seeking expert witness engagement
- **Plan Sponsors** — Fiduciary review and compliance consulting

## Ethical Standards

All materials adhere to **SER Best Practices**, including:

1. Truthfulness in rendering opinions
2. Competence verification before accepting engagements
3. Independence from case outcomes
4. Confidentiality of non-public documents
5. Professional conduct in disagreements
6. Accurate resume details (Best Practice #12)
